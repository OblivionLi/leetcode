Singly Linked List:

    Insertion Methods:

    insert_at_head(value): Inserts a node with the given value at the beginning of the list.
    insert_at_tail(value): Inserts a node with the given value at the end of the list.
    insert_after(node, value): Inserts a node with the given value after the specified node.
    insert_before(node, value): Inserts a node with the given value before the specified node.
    insert_at_index(index, value): Inserts a node with the given value at the specified index.
    Deletion Methods:

    delete_head(): Deletes the node at the head of the list.
    delete_tail(): Deletes the node at the tail of the list.
    delete_node(node): Deletes the specified node from the list.
    delete_at_index(index): Deletes the node at the specified index.
    Traversal Methods:

    traverse(): Traverses the entire list and prints or returns the elements.
    get_at_index(index): Returns the value of the node at the specified index.
    Search Methods:

    search(value): Searches for a node containing the given value and returns its index or reference.
    contains(value): Checks if the list contains a node with the given value.
    Utility Methods:

    size(): Returns the number of nodes in the list.
    is_empty(): Checks if the list is empty.
    reverse(): Reverses the order of nodes in the list.
    clear(): Clears all elements from the list.
    Other Methods:

    merge(list): Merges the current list with another list.
    split(index): Splits the list into two at the specified index.
